Trees

A tree is a hierarchical, non-linear data structure consisting of nodes connected by edges. It starts with a root node and extends into subtrees of children nodes.

Types of Trees:

  • Binary Tree: Each node has at most two children.
  • Binary Search Tree (BST): A binary tree where left child nodes have lesser value, and right child nodes have greater value than their parent.
  • Heap: A specialized tree-based structure satisfying the heap property.

Applications:

  • Hierarchical Data Storage: File systems, organizational structures.
  • Searching and Sorting: Binary search trees for efficient searching.
  • Parser Implementation: Syntax trees in compilers.

Graphs

A graph is a collection of nodes (vertices) connected by edges. It can model various complex structures.

Types of Graphs:

  • Directed Graph (Digraph): Edges have a direction, indicating a one-way relationship.
  • Undirected Graph: Edges are bidirectional.
  • Weighted Graph: Edges carry weights representing cost, distance, or any quantitative measure.

Applications:

  • Networking: Represents computer networks and social networks.
  • Transportation: Models routes in maps for GPS navigation.
  • Algorithm Implementation: Used in algorithms like Dijkstra's for shortest path finding.
